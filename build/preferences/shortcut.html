Define your own shortcut to open StoryTime.

  • CMD + Enter to add a story based on what's in the search box (first word will be the project, the remainder the story name).
  • Enter to start a timer on the selected item, again to stop the timer if the selected story is already timing.
  • Up / Down Arrows to navigate the list.
  • Right / Left Arrows view story detail / return to the main list.
  • CMD + Delete to delete the selected item.
  • ESC to close the pop-up.
  • CMD + L to focus the search box.
